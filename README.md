# 🎹 PlayErsGames Piano Game

Простая интерактивная пианино-игра на Python с использованием библиотеки Pygame.

## 📝 Описание

Пианино с 7 клавишами, каждая из которых воспроизводит соответствующую ноту. Клавиши отображаются на экране и реагируют на нажатия.

## 🎮 Управление

| Клавиша | Нота |
|---------|------|
| **Q** | DO |
| **W** | RE |
| **E** | MI |
| **R** | FA |
| **T** | SOL |
| **Y** | LA |
| **U** | SI |

## 🚀 Запуск

### Требования

- Python 3.x
- Pygame

### Установка зависимостей

```bash
pip install pygame
```

### Запуск игры

```bash
python piano.py
```

## 🛠️ Сборка в исполняемый файл

Проект настроен для сборки через PyInstaller:

```bash
pyinstaller piano.spec
```

Или:

```bash
pyinstaller --onefile --windowed --add-data "notes;notes" piano.py
```
